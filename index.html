EASyQuart Tools Documentation#

The easyquart python package aimed to supplement the modelling workflows for BHE simulations with the open source platform OpenGeoSys, a finite element solver for coupled thermo-hydro-chemo-mechanical problems in the subsurface, and its collection of Python tools OpenGeoSysTools.
