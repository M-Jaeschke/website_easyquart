The `easyquart` python package aimed to supplement the modelling workflows for BHE simulations with the open source platform [OpenGeoSys](https://www.opengeosys.org), a finite element solver for coupled thermo-hydro-chemo-mechanical problems in the subsurface, and its collection of Python tools [OpenGeoSysTools](https://ogstools.opengeosys.org/stable/).

**Date:** {sub-ref}`today`

**Useful links:**  [Source repository](https://gitlab.dit.htwk-leipzig.de/max.jaeschke/easyquart) | [Issues](https://gitlab.dit.htwk-leipzig.de/max.jaeschke/easyquart/-/issues)
